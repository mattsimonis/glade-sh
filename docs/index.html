What it does

Every device. One terminal.

Roost runs centrally on your machine. Sessions are alive whether you're at your desk, on the couch, or away from home.

📱

Mobile PWA

Install via "Add to Home Screen" on iOS or Android. Full-screen, no browser chrome. Works over Tailscale from anywhere.

⌨️

Custom mobile keyboard

Key toolbar with Esc, Tab, Ctrl, Alt, arrows, and combos. Long-press to repeat. Fully configurable — drag keys to reorder.

🪟

Project isolation

Each project gets its own tmux session and ttyd instance. Sessions survive browser closes, network drops, and reboots.

📜

Session logging

Every session recorded automatically via tmux pipe-pane. Browse and full-text search logs from the History tab.

⚡

Command snippets

Save commands and fire them into the terminal with one tap. Synced across all devices via the API.

🔒

Private by default

Everything runs on your hardware. Remote access via Tailscale mesh VPN. No cloud. No accounts. Nothing leaves your machine.

🔁

Auto-reconnect

Reconnects automatically on network drop or app background. Force-reconnects after >5s backgrounded.

🎨

Catppuccin Mocha

Consistent theme across terminal, UI chrome, toolbar, and this site. Berkeley Mono Nerd Font optional.

Get started

Up in minutes.

You need Docker and an always-on host. Everything else is handled by the Makefile.

1

Clone and configure

Copy the environment template. Set your hostname, domain, and dev directory.

git clone https://github.com/mattsimonis/roost && cp .env.example .env
2

DNS and TLS

Add a local DNS record in Pi-hole for ai.home. Generate a cert with mkcert and add the Caddyfile block to your reverse proxy. Full steps in SETUP.md.

3

Start

One command builds and starts both containers.

make setup
4

Open and install the PWA

Navigate to https://ai.home on any device. On iOS, tap Share → Add to Home Screen. Create your first project and the terminal opens.

Full walkthrough → SETUP.md

Stack

Solid foundations.

No novel dependencies. Every component is a proven, widely-used open source project.

ttyd — terminal over WebSocket tmux — session multiplexer Caddy — reverse proxy + TLS Tailscale — mesh VPN Docker Compose — container runtime Python stdlib — REST API, zero deps SQLite + FTS5 — embedded database xterm.js — terminal renderer Catppuccin Mocha — theme Oh My Zsh + Spaceship — shell
